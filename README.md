# django-vue-same-template

1. Start virtual environment
```
python -m venv venv
source venv/bin/activate
```

2. Install python libs
```
pip install -r requirements.txt
```

3. Do migrations
```
python ./manage.py migrate
```


4. Install npm modules
```
npm install
```

5. Run server
```
python ./manage.py runserver
```
